1. Описание выполнения:

1.1. Анализ задачи
Был дан одномерный массив размерности N. Требуется найти сумму 
отрицательных элементов между максимальным и минимальным элементами.
Уточнено: берутся первые вхождения min и max, сами они не учитываются.

1.2. Алгоритм
- Найти индексы первого вхождения min и max
- Определить левую границу = min(индекс_min, индекс_max) + 1
- Определить правую границу = max(индекс_min, индекс_max)
- Просуммировать отрицательные элементы на этом интервале

1.3. Реализация
Написан код на Python с функцией find_sum_between_extremes(arr).
Добавлена обработка случаев: массив менее 2 элементов, 
отсутствие элементов между min и max, отсутствие отрицательных.

1.4. Тестирование
Протестировано на следующих наборах:
- [10, -2, -5, 3, -8] → -7
- [1, 2, 3, 4] → 0
- [-5, -10, -3] → -5
- [3, -1, 5] → 0
- [5] → 0
Все тесты пройдены успешно.
